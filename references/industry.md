# 产业链分析模块

## 核心逻辑

**大趋势确认后 → 定位产业周期 → 量化市场规模 → 拆解产业链 → 找缺口（瓶颈/增量/国产替代）→ 分析利润分配 → 验证供需 → 催化剂排期 → 反面论证 → 筛选标的**

> 产业链上，哪里有缺口，就表示哪里有机会。

---

## 数据获取流程

```
Step 1: WebSearch 获取产业全貌
  → "[行业] 产业链 上游 中游 下游 全景图"
  → "[行业] 市场规模 2025 2030 预测 CAGR"
  → "[行业] 渗透率 当前 趋势"
  → "[行业] 发展阶段 成熟度"

Step 2: WebSearch 拆解各环节
  → "[行业] 竞争格局 龙头企业 市占率"
  → "[环节] 壁垒 技术门槛 进入难度"
  → "[产品] 价格走势 涨价 缺货 2025"
  → "[产品] 产能利用率 扩产计划 在建产能"
  → "[行业] 国产替代 自主可控 国产化率"
  → "[行业] 毛利率 净利率 各环节对比"

Step 3: 脚本获取各环节龙头财务数据
  → python3 scripts/run.py a <公司1> finance
  → python3 scripts/run.py a <公司2> finance
  → python3 scripts/run.py a <公司3> finance
  → 横向对比：谁营收增速最快？谁利润率最高？谁在加速？

Step 4: 脚本获取估值对比
  → python3 scripts/run.py a <公司> valuation
  → 哪个环节估值洼地？哪个已透支？

Step 5: 脚本获取板块整体走势
  → python3 scripts/run.py a <板块ETF> technical
  → 板块处于什么趋势阶段？是否已被充分定价？

Step 6: WebSearch 催化剂验证
  → "[行业] 政策 发布 预期 2025 2026"
  → "[公司] 新产品 发布时间 量产时间"
  → "[行业] 财报季 业绩预告"
```

## 第一步：产业生命周期定位

**任何分析前，先判断产业处于哪个阶段**：

| 阶段 | 特征 | 投资策略 | 关键指标 |
|------|------|----------|----------|
| **导入期** | 技术刚突破、产品未量产、参与者少 | 高风险高赔率，小仓位布局先驱者 | 融资事件、专利数、政策扶持 |
| **成长期** | 渗透率加速、收入爆发、新玩家涌入 | **最佳阶段**，找确定性最高的环节 | 营收增速>30%、渗透率10-50% |
| **成熟期** | 增速放缓、格局固化、比拼效率 | 只买龙头，赚稳定分红 | 行业增速<15%、CR5>60% |
| **衰退期** | 被替代、需求萎缩 | 回避，除非有转型逻辑 | 营收负增长、产能过剩 |

**判断方法**（搜索模板）：
```
"[行业] 发展阶段 行业周期"
"[行业] 市场规模 增速 CAGR"
"[行业] 渗透率 当前"
"[行业] 竞争格局 CR5 集中度"
```

---

## 第二步：市场规模量化

**必须回答**：这个市场有多大？增速多快？

| 指标 | 搜索方式 | 作用 |
|------|----------|------|
| TAM（全球市场规模） | `"[行业] global market size 2025 2030"` | 天花板有多高 |
| SAM（中国市场规模） | `"[行业] 中国 市场规模 2025"` | 实际可触达空间 |
| CAGR（复合增速） | `"[行业] CAGR 预测 forecast"` | 成长斜率 |
| 渗透率 | `"[行业] penetration rate 渗透率"` | 处于S曲线哪个位置 |

**市场规模评估标准**：
| 规模 | 吸引力 |
|------|--------|
| >1000亿人民币 | 大赛道，容纳多家大市值公司 |
| 200-1000亿 | 中等赛道，龙头有空间 |
| <200亿 | 小赛道，除非高速增长否则空间有限 |

## 第三步：渗透率 S 曲线定位

**渗透率决定了你在投什么阶段的机会**：

```
                        ┌─────────────────┐
                   ●●●● │   成熟期 >60%    │  增速放缓，赚效率
              ●●●●      └─────────────────┘
         ●●●●           ┌─────────────────┐
      ●●●               │ 加速期 10-60%    │  ★最佳投资窗口
    ●●                  └─────────────────┘
   ●                    ┌─────────────────┐
  ●                     │  早期 <10%       │  高赔率但不确定
 ●                      └─────────────────┘
```

| 阶段 | 渗透率 | 特点 | 投资策略 |
|------|--------|------|----------|
| 早期 | <10% | 技术路线未定、商业模式未验证 | 小仓位、看催化剂 |
| **加速期** | **10-30%** | **拐点刚过、增速最快、确定性提升** | **重点布局** |
| 中后期 | 30-60% | 仍在增长但边际放缓、竞争加剧 | 只买龙头 |
| 成熟期 | >60% | 格局固化、增量有限 | 赚分红，不赚增长 |

**搜索方式**：
```
"[产品/技术] 渗透率 2024 2025"
"[产品] penetration rate China"
"[产品] 出货量 总市场 比例"
```

## 第四步：产业链拆解

### 拆解三问法

1. 这东西要落地，**物理上需要什么？**（硬件/设备/材料）
2. 这东西要运转，**逻辑上需要什么？**（软件/服务/运营）
3. 这东西要扩张，**瓶颈在哪里？**（供给最紧的环节 = 缺口 = 机会）

### 标准结构

```
上游（原材料/设备/零部件）→ 中游（制造/集成）→ 下游（应用/客户）→ 配套（基础设施/软件/服务）
```

---

## 第五步：找缺口（三类机会）

### 类型一：瓶颈型（供给跟不上需求）

| 信号 | 含义 | 搜索验证 |
|------|------|----------|
| 产品涨价 | 供不应求 | `"[产品] 涨价 2025"` |
| 交期拉长 | 订单排满 | `"[产品] 交期 排产 2025"` |
| 产能利用率>90% | 满产 | `"[公司] 产能利用率"` |
| 扩产周期长（>2年） | 短期无法增产 | `"[产品] 新建产能 投产时间"` |
| 只有少数玩家能做 | 技术壁垒 | `"[产品] 全球供应商 市占率"` |

### 类型二：增量型（用量暴增）

| 信号 | 含义 | 搜索验证 |
|------|------|----------|
| 单位用量提升 | 技术升级带来更多需求 | `"[产品] 单机用量 提升"` |
| 渗透率加速 | S曲线拐点 | `"[产品] 渗透率 增速"` |
| 下游爆发 | 终端市场快速扩大 | `"[下游] 出货量 增长 2025"` |
| 新应用场景出现 | 增量需求凭空产生 | `"[产品] 新应用 新场景"` |

### 类型三：国产替代型

| 子类型 | 特征 | 搜索验证 | 投资节奏 |
|--------|------|----------|----------|
| **卡脖子型** | 被制裁/限制，必须替代 | `"[产品] 卡脖子 制裁 国产化"` | 政策驱动，确定性最高 |
| **性价比型** | 国产便宜好用，自然替代 | `"[产品] 国产 vs 进口 性价比"` | 市场驱动，看产品力 |
| **政策驱动型** | 信创/安全要求必须国产 | `"[领域] 信创 国产化要求"` | 看采购节奏 |

**国产替代阶段**：
```
0-5% 概念验证 → 5-20% 导入客户 → 20-50% 加速替代(★) → 50%+ 格局确立
```

## 第六步：利润分配分析（微笑曲线）

**谁赚最多钱？利润正在向哪里转移？**

```
利润率
  高│  品牌/IP        设计/研发
    │    ●               ●
    │     ●             ●
    │      ●           ●
    │       ●    制造  ●
  低│        ●●●●●●●●●
    └──────────────────────────
      上游          中游          下游
```

**分析维度**：

| 问题 | 搜索方式 |
|------|----------|
| 各环节毛利率对比 | `"[行业] 各环节 毛利率 对比"` |
| 谁有定价权 | `"[环节] 提价 转嫁成本"` |
| 利润在向哪转移 | `"[行业] 利润 转移 趋势"` |
| 哪个环节在降价 | `"[产品] 降价 趋势 价格战"` |

**利润转移方向判断**：
- 技术壁垒高的环节 → 利润集中（如芯片设计）
- 产能过剩的环节 → 利润被挤压（如组装代工）
- 掌握终端客户的环节 → 议价权强

## 第七步：环节综合评分

| 维度 | 1分 | 3分 | 5分 | 权重 |
|------|-----|-----|-----|------|
| 需求增速 | <10% | 20-30% | >50% | 25% |
| 供给约束/缺口 | 易扩产 | 1年建设期 | >2年+技术壁垒 | 25% |
| 竞争格局 | 极分散(CR5<20%) | 3-5家(CR5 40-60%) | 寡头垄断(CR5>70%) | 20% |
| 利润率 | 毛利<20% | 毛利30-50% | 毛利>50% | 15% |
| 国产替代空间 | 国产化率>70% | 30-70% | <30%（大量替代空间） | 15% |

**总分**（加权）：
- **> 4.0**：核心机会环节，立即深入
- **3.0-4.0**：值得跟踪
- **< 3.0**：优先级低

## 第八步：催化剂日历

**回答"什么时候兑现"**——列出未来 3-6 个月的关键事件：

| 时间类型 | 事件类型 | 搜索方式 |
|----------|----------|----------|
| 确定时间 | 财报发布 | `"[公司] 业绩预告 发布时间"` |
| 确定时间 | 政策落地 | `"[领域] 政策 实施时间"` |
| 大致时间 | 新产品上市 | `"[公司] 新产品 发布 量产 时间"` |
| 大致时间 | 产能投产 | `"[公司] 新产能 投产 达产"` |
| 不确定 | 订单落地 | `"[公司] 大客户 中标 签约"` |
| 不确定 | 技术突破 | `"[技术] 进展 突破 时间表"` |

**输出格式**：
```markdown
### 催化剂日历
| 时间 | 事件 | 影响环节 | 确定性 |
|------|------|----------|--------|
| 2025Q3 | XX公司新产品量产 | 上游材料 | 高 |
| 2025Q4 | 行业新标准实施 | 全链 | 高 |
| 2026H1 | XX公司产能释放 | 中游制造 | 中 |
```

## 第九步：反面论证（强制）

**每个"受益环节"必须附带反面论证**：

| 反面维度 | 问题 | 搜索方式 |
|----------|------|----------|
| 需求证伪 | 下游真的需要这么多吗？ | `"[下游] 实际需求 vs 预期"` |
| 替代风险 | 有没有更便宜的替代方案？ | `"[产品] 替代技术 新方案"` |
| 供给反转 | 扩产潮会不会导致过剩？ | `"[行业] 产能过剩 风险"` |
| 政策变化 | 补贴退出/政策转向？ | `"[行业] 补贴 退坡 政策变化"` |
| 竞争恶化 | 新玩家大量进入？ | `"[行业] 新进入者 竞争加剧"` |
| 估值透支 | 市场是否已经 price in？ | 通过脚本获取 PE 对比历史 |

**格式**：
```markdown
### 反面论证 / 这个逻辑哪里可能错
1. [风险点] — [可能性：高/中/低] — [验证信号：xxx]
2. ...
```
