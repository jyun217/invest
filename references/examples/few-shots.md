## 示例四：数据获取失败的标准处理

当脚本返回 error 时的写法：

```markdown
#### 财务表现
| 指标 | 数值 | 达标 | 来源 |
|------|------|------|------|
| 年收入 | 2,450亿 | ✓ | [腾讯/新浪API, 2026-06] |
| 收入增速 | 18% | ✓ | [腾讯/新浪API, 2026-06] |
| EBIT Margin | 22.5% | ✓ | [腾讯/新浪API, 2026-06] |
| 经营现金流 | 正 | ✓ | [腾讯/新浪API, 2026-06] |
| 自由现金流 | N/A（获取失败：脚本返回 error） | — | — |

> 注：自由现金流数据未能获取，公司质地评分取保守值（若数据完整可能更高）。
```
